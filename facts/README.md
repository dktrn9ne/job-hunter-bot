# Fact Bank

Truth-only, reusable facts used by the tailoring engine.

## Rules
- Only include claims you can defend (dates, employers, tools, metrics).
- Prefer measurable outcomes, but do not invent numbers.
- Keep bullets atomic (one claim per bullet).

## Files
- `marketing.json`
- `tech.json`
- `music.json`

Each file uses the same schema.
